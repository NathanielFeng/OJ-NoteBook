输入两个非负 10 进制整数 *A* 和 *B* (≤230−1)，输出 *A*+*B* 的 *D* (1<*D*≤10)进制数。

#### 输入格式

输入在一行中依次给出 3 个整数 *A*、*B* 和 *D*。

#### 输出格式

输出 *A*+*B* 的 *D* 进制数。

#### 输入样例

```in
123 456 8
```

#### 输出样例

```out
1103
```

#### 思路

参考10进制转2进制，采用除D取余法，当余数为0时结束~

注意：A = B = 0时的特殊情况，我自己就没考虑到~

```c++
#include<iostream>
#include<vector>
using namespace std;
int main() {
    int A, B, D, num;
    cin >> A >> B >> D;
    num = A + B;
    if(num == 0) {
        cout << 0;
        return 0;
    }
    vector<int> arr;
    while (num != 0) {
        arr.push_back(num % D);
        num /= D;
    }
    for (int i = arr.size() - 1; i >= 0; i--)
        cout << arr[i];
    return 0;
}
```
